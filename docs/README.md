# Project Documentation

This folder holds supporting material for the Personal Assistant Agent.

## Contents

- `screenshots/`: dashboard captures for English, Hebrew, desktop, and mobile layouts.

## Documentation Rules

- Never include OAuth tokens, API keys, email addresses, or private calendar details in screenshots.
- Use demo mode when creating public screenshots.
- Review screenshots before publishing them outside the project.
- Document any production integration changes alongside the related provider adapter.
